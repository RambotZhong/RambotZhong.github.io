Sublime compile tex not working

Problem

When using Sublime Text 2 + MacTex + Skim for the tex editing on mac, after all the setting, the compile of the sublime not working(using cmd + B, not working)

Solve

At sublime text, go to: Preferences -> Package Settings -> Settings - User, and reset the package to the default, then the compile works.
